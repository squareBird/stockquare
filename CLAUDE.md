# Stockquare

Stock trading system powered by KIS Open API.

## Rules

Before starting any work, you MUST read and follow `.aicontext/guides/GOLDEN_RULE.md`. This is the supreme rule of this project and overrides all defaults.

## Context

- `.aicontext/guides/` — Project guides (overview, git convention, project structure)
- `.aicontext/patterns/` — Code style and implementation patterns (backend / frontend)
- `.aicontext/spec/` — Feature specifications (backend / frontend)
- `.aicontext/decisions/` — Architecture decision records

## Tech Stack

- Backend: FastAPI + Python 3.12
- Frontend: Next.js + TypeScript
- Database: PostgreSQL
- Cache: Redis
- API: KIS Open API (Korea Investment & Securities)

## Language

- All code, comments, documentation, and commit messages in English.
- Respond to the user in Korean.
